## 1. Elicitation prompts

- “Before I explain, what would you predict and why?”
- “Show the first step you are confident about.”
- “What would have to be true for your explanation to work?”
- When the learner gives a model, ask which next case would challenge its boundary.

Use one prompt, not a questionnaire.

## 2. Gap probes

- Mechanism: “What changes if we remove this component?”
- Prerequisite: “Can you use X here without looking it up?”
- Discrimination: “Why is this X rather than the similar-looking Y?”
- Boundary: “Where does this rule stop working?”
- Metacognition: “Which sentence feels familiar but you could not derive?”

## 3. Retrieval formats

- Blank-page reconstruction.
- Teach it back to a beginner.
- Predict before revealing the result.
- Correct a plausible wrong explanation.
- Solve a near-neighbor case, then a transfer case.

Always close the loop with correction and reasoning.

## 4. Reflection and checkpoint

End a durable session with:

1. “Without looking back, what changed in your model today?”
2. “What can you now do that you could not do at the start?”
3. “What is still uncertain?”
4. Save one next move and, if valuable, one later retrieval probe.

## 5. Practice and review design

Build small sets around evidence:

- one retrieval item for the load-bearing idea;
- one discrimination item for the most likely confusion;
- one application item;
- one transfer item only after application succeeds.

Put missed items back into the next set. Increase spacing after successful
retrieval; shorten it after a miss. Avoid promising an exact universal schedule.

For a learning plan, record:

~~~text
Goal:
Current evidence:
Next capability:
Practice:
Mastery evidence:
Next probe:
~~~

## 6. Background explanations

Use these only when the learner needs a reason to engage with the method:

- Re-reading increases familiarity, which can be mistaken for recall.
- Curiosity and relevance make active engagement easier to sustain.
- Memory supports multi-step understanding; it is not the enemy of understanding.
- Mental rehearsal mainly strengthens patterns already practiced; it does not
  replace first-hand attempts at a new skill.

Keep mechanism claims modest. The runtime purpose is to select a better teaching
move, not to deliver a neuroscience lecture.
